සංයුත්තනිකායො

සගාථ වර්ගය

4. මාර සංයුත්තය

1. පළමු ( ආයු ) වග

4. ( පළමුවන ) පාස සූත්‍රය

මා විසින් මෙසේ අසන ලදී. එක් කලෙක භාග්‍යවතුන් වහන්සේ බරණැස් නුවර අසල ඉසිපතන් නම්වූ මුව වෙනෙහි වසනසේක. එහිදී භාග්‍යවත්හු ’’ මහණනි’’ යි භික්‍ෂූන්ට කථාකළසේක. ’’ස්වාමීන් වහන්සැ’’ යි ඒ භික්‍ෂූහු භාග්‍යවතුන් ට උත්තර දුන්හ.

භාග්‍යවතුන් වහන්සේ මෙසේ වදාළසේක: ’’ මහණනි, න්වණින් මෙනෙහි කිරීමෙන්, නුවණින් ප්‍රධාන වීර්‍ය්‍ය වැඩීමෙන් හැමටම උතුම් ( රහත්ඵල ) මිදුම මට ලැබිණ, හැමටම උතුම් ( රහත්ඵල ) මිදුම මවිසින් ප්‍රත්‍යක්‍ෂ කරණ ලදී. තෙපිද මහණනි, නුවණින් මෙනෙහි කිරීමෙන්, නුවණින් ප්‍රධාන වීර්‍ය්‍ය වැඩීමෙන් පරම ( රහත්ඵල ) මිදුම ලබවු, පරම මිදුම පැහැදිලි කරවු.’’

එවිට පාපී මාරයා භාග්‍යවතුන් වහන්සේ යම් තැනකද එතැනට පැමිණියේය. පැමිණ, භාග්‍යවතුන්ට ගාථායෙන් මෙසේ කීය:

’’මහණ තෝ කෙලෙස් මල පුඩුයෙන් බැඳුනෙහි යම් දෙව්සැප කෙනෙක් මිනිස් කම්සැප කෙනෙක් වෙත්ද, ඒ මාර බැම්මෙන් ( කෙලෙස් බැම්මෙන් ) බැඳුණෙහිය. මහණ, මගෙන් තෝ නොගැලවෙන්නෙහි.’’

’’ මාරය, මම් කෙලෙස් මලපුඩුයෙන් මිදුනෙමි. යම් දෙව් කම්සැප කෙනෙක් මිනිස් කම්සැප කෙනෙක් වෙද්ද, ඒ හැම මලපුඩුයෙන්ම මිදුනෙමි. මාරය තෝ නැසුණෙහිය.’’

එවිට පාපී මාරතෙම භාග්‍යවතුන් වහන්සේ තමා හඳුනතැයි, දනිතැයි, දැන, දුක් දොම්නස් පත්ව එහිම අතුරුදහන් විය.
