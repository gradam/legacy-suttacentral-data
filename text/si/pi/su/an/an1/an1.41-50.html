අඞ්ගුත්තර නිකාය

එක නිපාතය

5. පණිහිත (නොහොත් අච්ඡන්න) වර්‍ගය.

41–50.

මිච්ඡාපණිහිතචිත්ත සූත්‍රය.

’’මහණෙනි, ඇල්වී නණ්ඩුවක් හෝ යටවී නණ්ඩුවක් හෝ වරදවා ( තුඩ යටිකුරු කොට, නො ඇණෙන සේ ) තබන ලදුව අතින් හෝ පයින් හෝ මඬනා ලද්දේ අත හෝ පය හෝ නොබිඳෙන්නේය. ලේ හෝ උපදවන්නේයයි ඒ කාරණය විද්‍යාමාන නොවේ. මක්නිසාදයත්, මහණෙනි, නණ්ඩුව ( නොඇණෙන ලෙස ) වරදවා තබන ලද හෙයිනි. ’’මහණෙනි, එපරිද්දෙන්ම ඒකාන්තයෙන් යම් භික්‍ෂුවක් වරදවා තබන ලද සිතින් ’අවිද්‍යාව බිඳින්නේය. විද්‍යාව උපදවන්නේය. නිර්‍වාණය ප්‍රත්‍යක්‍ෂ කරන්නේයයි’ ( කියතොත් ) ඒ කාරණය ඇත්තක් නොවේ. මක්නිසාදයත්, මහණෙනි, සිත වරදවා යොදන ලද හෙයිනි.’’

සම්මාපණිහිත සූත්‍රය.

’’මහණෙනි, ඇල්වී නණ්ඩුවක් හෝ යටවී නණ්ඩුවක් හෝ තබන ලදුව අතින් හෝ පයින් හෝ මඬනා ලද්දේ අත හෝ පය හෝ බිඳින්නේය. ලේ හෝ උපදවන්නේය යන කාරණය සත්‍යයක් වන්නේය. ඊට හේතු කවරේද? මහණෙනි, නණ්ඩුව මනාකොට තබන ලද හෙයිනි. ’’මහණෙනි, එපරිද්දෙන්ම ඒකාන්තයෙන් යම් මහණෙක් තෙම මනාකොට තබන ලද සිතින් ’මහා අවිද්‍යාව බිඳින්නේය. විද්‍යාව උපදවන්නේය. නිර්‍වාණය ප්‍රත්‍යක්‍ෂ කරන්නේයයි’ ඒ කාරණය ඇත්තක් වන්නේය. ඊට හේතු කවරේද? මහණෙනි, ඔහුගේ සිත මනාකොට පිහිටුවන ලද හෙයිනි.’’

පදුට්ඨචිත්ත සූත්‍රය.

’’මහණෙනි, මේ ලෝකයෙහි ඉතා දූෂිතවූ ඇතැම් පුද්ගලයෙකුගේ සිත මම මාගේ සිතින් මෙසේ පිරිසිඳ දනිමි. මෙසමයෙහි මේ පුද්ගලයා ඉදින් කලුරිය කෙළේ නම් ( අතින් ) ගෙන එන ලද්දක් බහා තබන්නාක් මෙන් හෙතෙම නරකයෙහි උපදනේය. මක්නිසාදයත්, ඔහුගේ සිත ඉතා දූෂිතවූ හෙයිනි. ’’මහණෙනි, සිත බොහෝ සේ කෙලෙසීම හේතුකොට ගෙණද මේ ලෝකයෙහි ඇතැම් සත්ත්‍වයෝ ශරීරයාගේ භේදයෙන්, මරණින් මතු අපාය නම්වූ, දුගති නම්වූ, විනිපාත නම්වූ නරකයෙහි උපදිත්.’’

පසන්තචිත්ත සූත්‍රය.

’’මහණෙනි, මේ ලෝකයෙහි ප්‍රසන්නවූ සිත් ඇති ඇතැම් පුද්ගලයෙකුගේ සිත මම මාගේ සිතින් මෙසේ පිරිසිඳ දනිමි. ඉදින් මෙසමයෙහි මේ පුද්ගල තෙම කාලක්‍රියා කෙළේ නම් ගෙන එන ලද්දක් යම්සේ බහා තබනු ලැබේද, එසේම මෙතෙම ස්වර්‍ගයෙහි පිහිටියෙක් වේය කියායි. ඊට හේතු කවරේද? මහණෙනි, ඔහුගේ සිත ප්‍රසන්න හෙයිනි. මහණෙනි, සිතෙහි පැහැදීම හේතුකොට ගෙණද මේ ලෝකයෙහි සමහර සත්ත්‍වයෝ මෙසේ ශරීරයාගේ භේදයෙන්, මරණින් මතු සුගති නම්වූ ලෝකයෙහි උපදිත්.’’

ආචිලචිත්ත සූත්‍රය.

’’මහණෙනි, නොපැහැදුනාවූ, කැළඹුණු, කලල්වූ, මඩවූ යම් දියවිලක් වේද, ඇස් ඇති පුරුෂයෙක් තෙම ඉවුරෙහි සිටියේ, විලෙහි ඔබ මොබ හැසිරෙන්නාවූද, නැවතී සිටින්නාවූද සිප්පිබෙල්ලන්ද, සක්බෙල්ලන්ද, කැට හා කැබැලිතිද, මත්ස්‍ය සමූහයාද නොදක්නේය. ඊට හේතු කවරේද? මහණෙනි, ජලය නොපැහැදුනු බවයි. ’’මහණෙනි, එපරිද්දෙන්ම ඒකාන්තයෙන් යම් භික්‍ෂුවක් තෙම නොපැහැදුනාවූ සිතින් යුක්තව ආත්මාත්‍ර්‍ථය හෝ දැනගන්නේය. පරාත්‍ර්‍ථය හෝ දැන ගන්නේය. උභයාර්ථය හෝ දැනගන්නේය. මනුෂ්‍යධර්‍මයෙන් මත්තෙහිවූ ආර්‍ය්‍යභාව කරණයට ( අර්‍හත්බව සැලසීමට ) සමත්‍ර්‍ථවූ ඥානදර්‍ශනවිෂයෙක් හෝ ප්‍රත්‍යක්‍ෂ කරන්නේයයි ( කියතොත් ) ඒ කාරණය විද්‍යමාන නොවේ. ඊට හේතු කවරේද? මහණෙනි, ( ඔහුගේ ) සිත නොපැහැදුනු බවයි.’’

අනාචිලචිත්ත සූත්‍රය.

’’මහණෙනි, පිරිසිදුවූ, අතිශයින් ප්‍රසන්නවූ, නොකැළඹුනාවූ යම් දියවිලක් වේද, ඇස් ඇති පුරුෂයෙක් තෙම ඒ විල ඉවුරෙහි සිටියේ, විලෙහි ඔබ මොබ හැසිරෙන්නාවූද, නැවතී සිටින්නාවූද, සිප්පිබෙල්ලන් හා සක්බෙල්ලන්ද, කැට හා කැබැලිතිද, මත්ස්‍ය සමූහයාද දක්නේය. ඊට හේතු කවරේද? මහණෙනි, ජලය පැහැදුනු බවයි. ’’මහණෙනි, එපරිද්දෙන්ම ඒකාන්තයෙන් යම් මහටණක් තෙම පැහැදුනාවූ සිතින් යුක්තව ආත්මාත්‍ර්‍ථය හෝ දැනගන්නේය. පරාත්‍ර්‍ථය හෝ දැන ගන්නේය. උභයාර්ථය හෝ දැන ගන්නේය. මනුෂ්‍යධර්‍මයෙන් මත්තෙහිවූ ආර්‍ය්‍යභාව කරණයට ( අර්‍හත්බව සැලසීමට ) සමත්‍ර්‍ථවූ ඥානදර්‍ශන විශෙෂයක් හෝ ප්‍රත්‍යක්‍ෂ කරන්නේයයි ( කියතොත් ) ඒ කාරණය විද්‍යමාන වේ. ඊට හේතු කවරේද? මහණෙනි, ( ඔහුගේ ) චිත්තය පැහැදුනු බවයි.’’

මුදුචිත්ත ( අථවා පන්‍දන ) සූත්‍රය.

’’මහණෙනි, යම් කිසි වෘක්‍ෂ වර්‍ගයෝ වෙත්ද, ඔවුන් අතුරෙන් මෘදු බැවින් හා කටයුත්තකට සුදුසු බැවින් කොළොම් ගස යම් සේ අග්‍රයයි කියනු ලැබේද, මහණෙනි, එපරිද්දෙන්ම වඩනලද, නැවත නැවත පුරුදු කරණ ලද සිත හා සමාන මෘදුබව හා කටයුත්තට සුදුසු බව ඇති අනික් එකම ස්වභාවයකුදු මම නොම දකිමි. මහණෙනි, වඩනලද, නැවත නැවත පුරුදු කරණ ලද සිත මෘදුවද වේ. කටයුත්තට සුදුසුද වේ.’’

ලහුපරිචිත්ත සූත්‍රය.

’’මහණෙනි, යම්සේ සිත වහා ඉපිද වහා නැසෙන ස්වභාවද එබඳුවූ අනික් එකම ස්වභාවයකුදු මම නොම දකිමි. මහණෙනි, සිත කොතරම් වහා ඉපිද වහා නස්නා සවභාවද යත්, ඊට උපමාවකුදු පහසු නොවේ.’’

උපක්තිලිට්ඨචිත්ත සූත්‍රය.

’’මහණෙනි, මේ සිත පැහැ විහිදුවන්නේය. එය වනාහි ආගන්තුකවූ ( රාගාදී ) උපක්ලේශයන් විසින් කිලිටි කරණ ලද්දේය.’’

විප්පමුත්තචිත්ත සූත්‍රය.

’’මහණෙනි, මේ සිත වනාහි ආගන්තුකවූ ( රාගාදී ) උපක්ලෙශයන්ගෙන් හාත් පසින් මිදුනේය.’’

පස්වන වර්‍ගය නිමි.
