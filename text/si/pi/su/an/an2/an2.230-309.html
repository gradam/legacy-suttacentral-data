3. තුන්වෙනි පණ්ණාසකය

17. විනය පෙය්‍යාල හෙවත් ශික්ෂාපද වර්ගය

සික්ඛාපද පඤ්ඤප්ති සූත්‍රය:—1. “මහණෙනි, මේ දෙක අනුසස් සඳහා තථාගතයන් විසින් ශ්‍රාවකයන්හට ශික්ෂාපද පනවනලදී. කවර දෙකක් සඳහාද යත්:

2. ‘මැනව, වහන්සැ’යි සංඝයාගේ පිළිගැනීම පිණිසද සංඝයාගේ පහසු බව පිණිසද යන දෙක අනුසස් සඳහායි

3. “මහණෙනි, මේ දෙක අනුසස් සඳහා තථාගතයන් විසින් ශ්‍රාවකයන්හට ශික්ෂාපද පනවන ලද්දේය.”

(2–10 සූත්‍රවල 1, 3 ඡෙද මේ වර්ගයේ 1 සූත්‍රයේ 1, 3 ඡෙද මෙනි.)

2. “දුසිල් පුද්ගලයන්ට නිග්‍රහ පිණිසද ප්‍රියශීලී භික්ෂූන්ගේ පහසු විහරණය පිණිසද යන දෙක සඳහායි

2 “මේ ආත්මයෙහි කෙලෙසුන්ගේ සංවරය පිණිසද පරලෙව්හි කෙලෙසුන්ගේ වැලැක්ම පිණිසද යන දෙක සඳහායි.

2. “මේ ආත්මයෙහි වෛරයන්ගේ සංවරය පිණිසද පරලෙව්හි වෛරයන්ගේ නැසීම පිණිසද යන දෙක සඳහායි.

2. “මේ ආත්මයෙහි දොෂයන්ගේ සංවරය පිණිසද පරලෙව්හි දොෂයන්ගේ වැලැක්ම පිණිසද යන දෙක සඳහායි.

2. “මේ ආත්මයෙහි භය හේතූන්ගේ සංවරය පිණිසද පරලෙව්හි භය හේතූන්ගේ ප්‍රතිෂෙධය පිණිසද යන දෙක සඳහායි.

2. “මේ ආත්මයෙහි අකුසල ධර්මයන්ගේ සංවරය පිණිසද පරලෙව්හි අකුසල ධර්මයන්ගේ වැලැක්ම පිණිසද යන දෙක සඳහායි.

2 “ගිහිගෙයි වාසීන්ට අනුකම්පා පිණිසද පවිටු භික්ෂූන්ගේ පක්ෂබල සිඳීම පිණිසද යන දෙක සඳහායි.

2. “නොපහන් පුද්ගලයන්ගේ ප්‍රසාදය පිණිසද පහන් පුද්ගලයන්ගේ ප්‍රසාදය බොහෝ බව (වැඩීම) පිණිසද යන දෙක සඳහායි.

2. සද්ධර්මයාගේ (ශාසනයාගේ) චිරස්ථිතිය පිණිසද විනයට අනුග්‍රහය පිණිසද යන දෙක සඳහායි

පාතිමොක්ඛ පඤ්ඤත්ති සූත්‍ර.

1. “මහණෙනි, මේ දෙක අනුසසුන් සඳහා තථාගතයන් විසින් ශ්‍රාවකයන්හට පාමොක් පනවන ලදී. කවර දෙකක් සඳහාද යත්, 2. “මැනව, වහන්සැ’යි සංඝයාගේ පිළිගැනීම පිණිසද, සංඝයාගේ පහසු බව පිණිසද යන දෙක පිණිසයි. 3. “මහණෙනි, මේ දෙක අනුසස් සඳහා තථාගතයන් විසින් ශ්‍රාවකයන්හට පාමොක් පනවන ලද්දේය. (12–20 සූත්‍රවල 1, 3 ඡෙද මේ වර්ගයේ 11 සූත්‍රයේ 1, 3 ඡෙද මෙන් වෙත්.)

2. “දුසිල් පුද්ගලයන්ට නිග්‍රහ පිණිස හා ප්‍රියශීලී භික්ෂූන්ගේ පහසු විහරණය පිණිසයි.

2. “මේ ආත්මයෙහිවූ කෙලෙසුන්ගේ සංවරය පිණිස හා පරලෙව්හිවූ කෙලෙසුන්ගේ වැලැක්ම පිණිසයි.

2. “මේ ආත්මයෙහි වෛරයන්ගේ සංවරය පිණිස හා පරලෙව්හි වෛරයන්ගේ නැසීම පිණිසයි.

2. “මේ ආත්මයෙහිවූ දොෂයන්ගේ සංවරය පිණිස හා පරලෙව්හිවූ දොෂයන්ගේ වැලැක්ම පිණිසයි.

2. ‘මේ ආත්මයෙහිවූ භය හේතූන්ගේ සංවරය පිණිස හා පරලෙව්හිවූ භය හේතූන්ගේ ප්‍රතිෂෙධය පිණිසයි.

2. “මේ ආත්මයෙහිවූ අකුසල ධර්මයන්ගේ සංවරය පිණිස හා පරලෙව්හිවූ අකුසල ධර්මයන්ගේ වැලැක්ම පිණිසයි.

2. “ගිහිගෙයි වාසීන්ට අනුකම්පා පිණිස හා පවිටු භික්ෂූන්ගේ පක්ෂ බල සිඳීම පිණිසයි.

2. “නොපහන් පුද්ගලයන්ගේ ප්‍රසාදය පිණිස හා පහන් පුද්ගලයන්ගේ ප්‍රසාදය බොහෝ බව (වැඩීම) පිණිසයි.

2. “සද්ධර්මයාගේ (ශාසනයාගේ) චිරස්ථිතිය පිණිස හා විනයට අනුග්‍රහය පිණිසයි.

පාතිමොක්ඛුද්දේස සූත්‍ර

1. “මහණෙනි, මේ දෙක අනුසසුන් සඳහා තථාගතයන් විසින් ශ්‍රාවකයන්හට පාමොක් උදෙසීම පනවන ලදී. කවර දෙක සඳහාද යත්, 2. “මැනව, වහන්සැයි’ සංඝයාගේ පිළිගැනීම පිණිසද සංඝයාගේ පහසු බව පිණිසද යන දෙක අනුසස් පිණිසයි. 3. “මහණෙනි, මේ දෙක අනුසසුන් සඳහා තථාගතයන් විසින් ශ්‍රාවකයන්හට පාමොක් උදෙසීම පනවන ලද්දේය (22–30 මේ සූත්‍රවල 1, 3 ඡෙද මේ වර්ගයේ 21 වෙනි සූත්‍රයේ 1 3 ඡෙද මෙන් වෙත්.)

2. “දුසිල් පුද්ගලයන්ට නිග්‍රහ පිණිස හා ප්‍රියශීලී භික්ෂූන්ගේ පහසු විහරණය පිණිසයි.

2. “මේ ආත්මයෙහිවූ කෙලෙසුන්ගේ සංවරය පිණිස හා පරලෙව්හිවූ කෙලෙසුන්ගේ වැලැක්ම පිණිසයි.

2. ‘මේ ආත්මයෙහිවූ වෛරයන්ගේ සංවරය පිණිස හා පරලෙව්හිවූ වෛරයන්ගේ නැසීම පිණිසයි.

2. “මේ ආත්මයෙහිවූ භය හේතූන්ගේ සංවරය පිණිස හා පරලෙව්හිවූ භය හේතූන්ගේ නැසීම පිණිසයි.

2. “මේ ආත්මයෙහිවූ අකුසල ධර්මයන්ගේ සංවරය පිණිස හා පරලෙව්හිවූ අකුසල ධර්මයන්ගේ වැලැක්ම පිණිසයි.

2. “ගිහිගෙයි වාසීන්ට අනුකම්පා පිණිස හා පවිටු භික්ෂූන්ගේ පක්ෂබල සිඳීම පිණිසයි.

2. “නොපහන් පුද්ගලයන්ගේ ප්‍රසාදය පිණිස හා පහන් පුද්ගලයන්ගේ ප්‍රසාදය බොහෝ බව (වැඩීම) පිණිසයි.

2. “සද්ධර්මයාගේ (ශාසනයාගේ) චිරස්ථිතිය පිණිස හා විනයට අනුග්‍රහය පිණිසයි.

තිණවත්ථාරක සූත්‍ර

1. “මහණෙනි, මේ දෙක අනුසසුන් සඳහා තථාගතයන් විසින් (ගුරු ඇවැත් හා ගිහියන් සම්බන්ධ ඇවැත් හැර සෙසු ඇවතින් පිරිසිදුවීම පිණිස) තිණවත්ථාරක කර්මය ශ්‍රාවකයන්හට පනවන ලදී. කවර දෙකක් සඳහාද යත්, 2. “මැනව, වහන්සැයි’ සංඝයාගේ පිළිගැනීම පිණිස හා සංඝයාගේ පහසු බව පිණිසයි. 3. “මහණෙනි, මේ දෙක අනුසස් සඳහා තථාගතයන් විසින් (ගුරු ඇවැත් හා ගිහියන් සම්බන්ධ ඇවැත් හැර සෙසු ඇවතින් පිරිසිදුවීම පිණිස) තිණවත්ථාරක කර්මය ශ්‍රාවකයන්හට පනවන ලදී.” (292–300 සූත්‍රවල 1, 3 ඡෙද මේ වර්ගයේ 291 වෙනි සූත්‍රයේ 1, 3 ඡෙද මෙන් වෙත්.)

2. “දුසිල් පුද්ගලයන්ට නිග්‍රහ පිණිස හා ප්‍රියශීලී භික්ෂූන්ගේ පහසු විහරණය පිණිසයි.

2. “මේ ආත්මයෙහිවූ කෙලෙසුන්ගේ සංවරය පිණිස හා පරලෙව්හිවූ කෙලෙසුන්ගේ වැලැක්ම පිණිසයි.

2. “මේ ආත්මයෙහිවූ දොෂයන්ගේ සංවරය පිණිස හා පරලෙව්හිවූ දොෂයන්ගේ වැලැක්ම පිණිසයි.

2. “මේ ආත්මයෙහිවූ භය හේතූන්ගේ සංවරය පිණිස හා පරලෙව්හිවූ භය හේතූන්ගේ නැසීම පිණිසයි.
