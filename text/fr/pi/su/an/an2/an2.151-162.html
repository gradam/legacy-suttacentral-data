Aṅguttara Nikāya

151–162. Accueil

Il y a, bhikkhous, ces deux [types d']accueil. Quels sont ces deux? L'accueil avec les choses matérielles et l'accueil avec le Dhamma. Voici, bhikkhous, quels sont ces deux [types d']accueil. De ces deux [types d']accueil, le plus éminent est l'accueil avec le Dhamma.

Il y a, bhikkhous, ces deux [types de] service. Quels sont ces deux? Le service dans les choses matérielles et le service dans le Dhamma. Voici, bhikkhous, quels sont ces deux [types de] service. De ces deux [types de] service, le plus éminent est le service dans le Dhamma.

Il y a, bhikkhous, ces deux [types de] recherche. Quels sont ces deux? La recherche de choses matérielles et la recherche du Dhamma. Voici, bhikkhous, quels sont ces deux [types de] recherche. De ces deux [types de] recherche, le plus éminent est la recherche du Dhamma.

Il y a, bhikkhous, ces deux [types de] quête. Quels sont ces deux? La quête de choses matérielles et la quête du Dhamma. Voici, bhikkhous, quels sont ces deux [types de] quête. De ces deux [types de] quête, le plus éminent est la quête du Dhamma.

Il y a, bhikkhous, ces deux [types de] poursuite. Quels sont ces deux? La poursuite de choses matérielles et la poursuite du Dhamma. Voici, bhikkhous, quels sont ces deux [types de] poursuite. De ces deux [types de] poursuite, le plus éminent est la poursuite du Dhamma.

Il y a, bhikkhous, ces deux [types de] vénération. Quels sont ces deux? La vénération avec les choses matérielles et la vénération dans le Dhamma. Voici, bhikkhous, quels sont ces deux [types de] vénération. De ces deux [types de] vénération, le plus éminent est la vénération dans le Dhamma.

Il y a, bhikkhous, ces deux [types de] cadeau pour un invité. Quels sont ces deux? Un cadeau pour un invité avec les choses matérielles et un cadeau pour un invité dans le Dhamma. Voici, bhikkhous, quels sont ces deux [types de] cadeau pour un invité. De ces deux [types de] cadeau pour un invité, le plus éminent est le cadeau pour un invité dans le Dhamma.

Il y a, bhikkhous, ces deux [types de] prospérité. Quels sont ces deux? La prospérité dans les choses matérielles et la prospérité dans le Dhamma. Voici, bhikkhous, quels sont ces deux [types de] prospérité. De ces deux [types de] prospérité, le plus éminent est la prospérité dans le Dhamma.

Il y a, bhikkhous, ces deux [types de] croissance. Quels sont ces deux? La croissance dans les choses matérielles et la croissance dans le Dhamma. Voici, bhikkhous, quels sont ces deux [types de] croissance. De ces deux [types de] croissance, le plus éminent est la croissance dans le Dhamma.

Il y a, bhikkhous, ces deux [types de] joyau. Quels sont ces deux? Le joyau matériel et le joyau du Dhamma. Voici, bhikkhous, quels sont ces deux [types de] joyau. De ces deux [types de] joyau, le plus éminent est le joyau du Dhamma.

Il y a, bhikkhous, ces deux [types d']accumulation. Quels sont ces deux? L'accumulation de choses matérielles et l'accumulation du Dhamma. Voici, bhikkhous, quels sont ces deux [types d']accumulation. De ces deux [types d']accumulation, le plus éminent est l'accumulation du Dhamma.

Il y a, bhikkhous, ces deux [types de] plénitude. Quels sont ces deux? La plénitude dans les choses matérielles et la plénitude dans le Dhamma. Voici, bhikkhous, quels sont ces deux [types de] plénitude. De ces deux [types de] plénitude, le plus éminent est la plénitude dans le Dhamma.
