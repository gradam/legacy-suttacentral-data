Aṅguttara Nikāya

2. Efforts

Ces deux [types d']effort, bhikkhous, sont difficiles à réaliser dans le monde. Quels sont ces deux? L'effort des gens de foyer habitant à la maison dans le but d'offrir des robes, de la nourriture, des logements, des remèdes et des provisions pour les malades, et l'effort de ceux qui ont quitté la vie de foyer pour la vie sans foyer dans le but de renoncer à toutes les acquisitions. Ces deux [types d']effort, bhikkhous, sont difficiles à réaliser dans le monde. Parmi ces deux [types d']effort, l'effort dans le but de renoncer à toutes les acquisitions est le plus élevé.

C'est pourquoi, bhikkhous, vous devriez vous entraîner ainsi: 'Nous nous efforcerons dans le but de renoncer à toutes les acquisitions.' Voici, bhikkhous, comment vous devriez vous entraîner.
