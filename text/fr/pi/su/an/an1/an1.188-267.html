Aṅguttara Nikāya

188–267. Les disciples les plus distingués

Bhikkhous, parmi tous mes disciples, Aññāsikoṇḍañña fut celui qui parvint le premier à la réalisation.

Sāriputta est celui qui a la plus grande sagesse.

Mahāmoggallāna est éminent pour ses pouvoirs supranormaux.

Mahākassapa est éminent pour son observance des austérités.

Anuruddha est éminent pour son pouvoir de vision divine.

Le fils de Kāḷigodhāya est éminent pour sa naissance dans une famille de haut rang.

Lakuṇḍaka Bhaddiya est éminent pour son manque de charme.

Piṇḍola Bhāradvāja est éminent pour son rugissement de lion.

Puṇṇa Mantāṇiputta est éminent pour sa dispense de discours sur le Dhamma.

Mahākaccāna est éminent pour ses explications brèves.

Cūḷapanthaka est éminent pour sa création d'images mentales.

Cūḷapanthaka est éminent pour l'assujettissement de son esprit.

Mahāpanthaka est éminent pour son assujettissement des sensations.

Subhūti est éminent pour son séjour en forêt.

Subhūti est éminent pour sa recette d'offrandes.

Revata Khadiravaniya est éminent pour son gain de la perception de la forêt.

Kaṅkhārevata est éminent pour sa réalisation des jhānas.

Soṇa Koḷivisa est éminent pour son effort actif.

Soṇa Kuṭikaṇṇa est éminent pour sa parole polie.

Sīvali est éminent pour ses gains.

Vakkalī est éminent pour sa libération par la conviction.

Rāhula est éminent pour son désir de l'entraînement.

Raṭṭhapāla est éminent pour sa renonciation à la vie domestique sur la base de la conviction.

Kuṇḍadhāna est éminent pour la réception du premier morceau (de nourriture).

Vaṅgīsa est éminent pour son explication de l'Enseignement.

Upasena Vaṅgantaputta est éminent pour son amitié envers tous.

Dabba Mallaputta est éminent pour la préparation des lits et des sièges.

Pilindavaccha est éminent pour son adoration par les dévas.

Bāhiya Dārucīriya est éminent pour sa réalisation instantanée de l'Enseignement.

Kumārakassapa est éminent pour ses arrangements verbaux amicaux.

Mahākoṭṭhita est éminent pour sa connaissance analytique.

Ānanda est éminent pour son érudition.

Ānanda est éminent pour son attention.

Ānanda est éminent pour son comportement correct.

Ānanda est éminent pour son courage.

Ānanda est éminent pour son aide portée aux autres.

Uruvelakassapa est éminent pour avoir le plus grand nombre de disciples.

Kāḷudāyī est éminent pour la réconciliation des clans.

Bākula est éminent pour la rareté de ses maladies.

Sobhita est éminent pour les souvenirs de ses vies passées.

Upāli est éminent pour sa mémorisation de la discipline.

Nandaka est éminent pour ses conseils aux bhikkhunīs.

Nanda est éminent pour la protection de ses portes sensorielles.

Mahākappina est éminent pour ses conseils aux bhikkhous.

Sāgata est éminent pour l'entrée dans l'élément feu.

Rādha est éminent pour son explication de l'Enseignement.

Mogharājā est éminent pour le port de robes usées.

Bhikkhous, parmi mes disciples féminins, Mahāpajāpatigotamī fut la première à devenir arahant.

Khemā est éminente pour sa haute sagesse.

Uppalavaṇṇā est éminente pour ses pouvoirs psychiques.

Paṭācārā est éminente pour son maintien de la discipline.

Dhammadinnā est éminente pour son explication de l'Enseignement.

Nandā est éminente pour sa réalisation des jhānas.

Soṇā est éminente pour son effort actif.

Bakulā est éminente pour son pouvoir de vision divine.

Bhaddā Kuṇḍalakesā est éminente pour sa réalisation instantanée de l'Enseignement.

Bhaddā Kāpilānī est éminente pour les souvenirs de ses vies précédentes.

Bhaddakaccānā est celle qui a la plus grande sagesse.

Kisāgotamī est éminente pour le port de robes usées.

Siṅgālakamātāti est éminente pour sa libération par la conviction.

Bhikkhous, parmi mes disciples laïcs, Tapussa et Bhallikā les marchands furent les premiers à prendre refuge.

Anāthapiṇḍika, qui est l'homme de foyer Sudatta, est le plus fervent de mes disciples laïcs.

Citta Macchikāsaṇḍika l'homme de foyer est éminent pour son explication de l'Enseignement.

Hatthaka Āḷavaka est éminent pour son établissement de la libéralité, de la parole aimable, le fait de mener une vie utile, et son état d'équanimité lorsqu'il est parmi les autres.

Mahānāma le Sakka est éminent pour ses disciples distingués.

Ugga l'homme de foyer de Vésāli est le plus aimable.

Uggata l'homme de foyer d'Hatthigāma est celui qui s'occupe le mieux de la communauté.

Sūrambaṭṭha est éminent pour la réconciliation.

Jīvaka Komārabhacca est éminent pour son amabilité envers tous.

Nakulapitā l'homme de foyer est éminent pour les confidences.

Bhikkhous, parmi mes disciples laïcs féminins, Sujātā la fille de Seniya a été la première à prendre refuge.

Visākhā la mère de Migāra est la plus fervente de mes disciples féminins.

Khujjuttarā est la plus érudite.

Sāmāvatī est la meilleure pour développer la bienveillance.

Uttarānandamātā est éminente pour sa réalisation des jhānas.

Suppavāsā la fille des Koliya est la disciple la plus aimable.

Suppiyā est la meilleure pour s'occuper des malades.

Kātiyānī est éminente pour son amabilité permanente.

Nakulamātā la femme de l'homme de foyer est éminente pour son amabilité indivisée.

Kāḷī Kulagharikā est éminente pour l'adhérence aux ouï-dire.
