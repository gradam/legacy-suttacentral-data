Aṅguttara Nikāya

268–277. Impossible

Il est impossible, bhikkhous, cela ne peut se produire qu'un individu ayant atteint la vue correcte envisage un quelconque phénomène conditionné comme étant constant. Cela est impossible. Mais il est possible, bhikkhous, qu'un individu ordinaire envisage un certain phénomène conditionné comme étant constant. Cela est possible.

Il est impossible, bhikkhous, cela ne peut se produire qu'un individu ayant atteint la vue correcte envisage un quelconque phénomène conditionné comme étant agréable. Cela est impossible. Mais il est possible, bhikkhous, qu'un individu ordinaire envisage un certain phénomène conditionné comme étant agréable. Cela est possible.

Il est impossible, bhikkhous, cela ne peut se produire qu'un individu ayant atteint la vue correcte envisage un quelconque phénomène comme étant soi. Cela est impossible. Mais il est possible, bhikkhous, qu'un individu ordinaire envisage un certain phénomène comme étant soi. Cela est possible.

Il est impossible, bhikkhous, cela ne peut se produire qu'un individu ayant atteint la vue correcte prenne la vie [=tue] de sa mère. Cela est impossible. Mais il est possible, bhikkhous, qu'un individu ordinaire prenne la vie de sa mère. Cela est possible.

Il est impossible, bhikkhous, cela ne peut se produire qu'un individu ayant atteint la vue correcte prenne la vie de son père. Cela est impossible. Mais il est possible, bhikkhous, qu'un individu ordinaire prenne la vie de son père. Cela est possible.

Il est impossible, bhikkhous, cela ne peut se produire qu'un individu ayant atteint la vue correcte prenne la vie d'un arahant. Cela est impossible. Mais il est possible, bhikkhous, qu'un individu ordinaire prenne la vie d'un arahant. Cela est possible.

Il est impossible, bhikkhous, cela ne peut se produire qu'un individu ayant atteint la vue correcte fasse saigner le Tathāgata avec un esprit malveillant. Cela est impossible. Mais il est possible, bhikkhous, qu'un individu ordinaire fasse saigner le Tathāgata avec un esprit malveillant. Cela est possible.

Il est impossible, bhikkhous, cela ne peut se produire qu'un individu ayant atteint la vue correcte crée une scission du Saṅgha. Cela est impossible. Mais il est possible, bhikkhous, qu'un individu ordinaire crée une scission du Saṅgha. Cela est possible.

Il est impossible, bhikkhous, cela ne peut se produire qu'un individu ayant atteint la vue correcte recommande un autre enseignant. Cela est impossible. Mais il est possible, bhikkhous, qu'un individu ordinaire recommande un autre enseignant. Cela est possible.

Il est impossible, bhikkhous, cela ne peut se produire que deux arahants Bouddhas apparaissent en même temps dans le même système-monde. Cela est impossible. Mais il est possible, bhikkhous, qu'un seul arahant Bouddha apparaisse dans un système-monde. Cela est possible.
