Ангуттара Никая

Иттха сутта

10.73 Желанное

[Благословенный сказал]: «Монахи, есть десять вещей—желанных, приятных, и редко обретаемых в мире. Какие десять?

  1. Богатство—желанно, приятно и редко обретаемо в мире.
  2. Красота …
  3. Здоровье …
  4. Нравственное поведение …
  5. Целомудрие…
  6. Друзья…
  7. Учёность…
  8. Мудрость…
  9. Хорошие качества…
  10. Небесные миры—желанны, приятны и редко обретаемы в мире.

Таковы десять вещей—желанных, приятных, и редко обретаемых в мире. Есть эти [иные] десять вещей, монахи, которые являются преградой этим десяти вещам, которые желанны, приятны и редко обретаемы в мире:

  1. Лень и отсутствие инициативы являются преградой для богатства.
  2. Отсутствие украшения и наведения красоты являются преградой для красоты.
  3. Совершение не полезного является преградой для здоровья.
  4. Плохие друзья являются преградой для нравственного поведения.
  5. Отсутствие сдержанности органов чувств является преградой для целомудрия.
  6. Двуличность является преградой для дружбы.
  7. Отсутствие декламации [вслух по памяти] является преградой для учёности.
  8. Нежелание слушать и задавать вопросы является преградой для мудрости.
  9. Отсутствие усилия и недостаток рассмотрения являются преградой для хороших качеств.
  10. Неправильная практика является преградой для небесных миров.

Таковы [иные] десять вещей, монахи, которые являются преградой этим десяти вещам, которые желанны, приятны и редко обретаемы в мире.

Есть эти [ещё другие] десять вещей, монахи, которые являются питанием для этих десяти вещей, которые желанны, приятны и редко обретаемы в мире:

  1. Прилежание и инициатива являются питанием для богатства.
  2. Украшение и наведение красоты являются питанием для красоты.
  3. Совершение полезного является питанием для здоровья.
  4. Хорошие друзья являются питанием для нравственного поведения.
  5. Сдержанность органов чувств является питанием для целомудрия.
  6. Искренность является питанием для дружбы.
  7. Декламация [вслух по памяти] является питанием для учёности.
  8. Желание слушать и задавать вопросы является питанием для мудрости.
  9. Усилия и рассмотрение являются питанием для хороших качеств.
  10. Правильная практика является питанием для небесных миров.

Таковы [ещё другие] десять вещей, монахи, которые являются питанием для этих десяти вещей, которые желанны, приятны и редко обретаемы в мире».
