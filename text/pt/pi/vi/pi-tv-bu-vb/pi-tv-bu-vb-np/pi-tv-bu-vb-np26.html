COLEÇÃO DAS REGRAS MONÁSTICAS DO BUDISMO THERAVĀDA

O Código de Disciplina Monástica dos Bhikkhus

Regras Nissaggiya

26. Pedir por linha de costura

História de origem

Em certa ocasião o Abençoado estava em Rajagaha no Bambual, no Santuário dos Esquilos. From the origin story to pācittiya 32 it seems cīvarakārasamaya refers to any time one is making a robe, see MS.2.729 . Naquela ocasião, bhikkhus do Grupo dos Seis faziam mantos, e ele pediram por uma grande quantidade de linha de costura. E assim, após o mantos terem sido costurados houve sobra de uma grande quantidade de linha. Aqueles bhikkhus disseram: “Pois bem, vamos pedir por mais linha e então arranjar que com esta tecido para mantos seja feito para nós.” E assim eles procederam. E assim, após o tecido para mantos ter sido costurado houve sobra de uma grande quantidade de linha. Eles então pediram por mais linha uma segunda vez e arranjaram que com esta tecido para mantos fosse feito para eles. E assim, após o tecido para mantos ter sido costurado houve sobra de uma grande quantidade de linha. Eles então pediram por mais linha uma terceira vez e arranjaram que com esta tecido para mantos fosse feito para eles. As pessoas se queixaram e criticaram: “Como podem estes contemplativos Sákya pedir por linha e então arranjar que com esta tecido para mantos seja feito para eles?”

Os bhikkhus ouviram as queixas e críticas daquelas pessoas, e os bhikkhus de poucos desejos ... se queixaram e criticaram aqueles bhikkhus: “Como podem estes bhikkhus do Grupo dos Seis pedir por linha e então arranjar que com esta tecido para mantos seja feito para eles?”

Após criticar os bhikkhus do Grupo dos Seis de várias maneiras, aqueles bhikkhus informaram o Abençoado o ocorrido ... “É verdade, bhikkhus, que vocês fizeram isto?”

“É verdade, mestre.”

O Buda então os repreendeu: “… “Homens tolos, como puderam vocês pedir por linha e então arranjar que com esta tecido para mantos fosse feito para vocês?” Isto não fará surgir fé naqueles ainda sem fé … E então, bhikkhus, esta regra de treinamento deve ser assim recitada:

Regra final

“Se um bhikkhu pede por linha e em seguida arranja que com esta seja feito tecido para mantos, ele comete uma ofensa nissaggiya pacittiya, que implica a confissão com renúncia.

Definições

“Um”: refere-se a qualquer indivíduo, de tal tipo, de tal ocupação, de tal status social, de tal nome, de tal família, de tal conduta, de tal comportamento, de tal associação, seja de muita ou pouca idade, ou não pertencendo a nenhum grupo específico. … “Bhikkhu”: … neste caso, um bhikkhu é alguém que tenha recebido a ordenação completa na Sangha de bhikkhus através de um procedimento formal, consistindo de três moções e três proclamações, que é irrefutável e completo. Isto se entende por bhikkhu.

“Ele mesmo”: o próprio bhikkhu é quem pede.

“Linha”: há seis tipos de linha: de linho, de algodão, de seda, de lã, de juta, de cânhamo.

“Costura”: para cada ação necessária para que o tecido seja costurado, ele comete uma ofensa dukkata, de transgressão. Quando ele obtém o tecido, a posse deste deverá ser renunciada.

E a posse deste deverá ser renunciada e transferida para a Sangha dos bhikkhus, um grupo de bhikkhus ou um determinado bhikkhu. “E, bhikkhus, isto deverá ser renunciado desta maneira:” “… “… (É expandido como na Regra Nissaggiya 1, com as devidas substituições) … ‘Veneráveis, este tecido para manto que arranjei que fosse feito com linha por mim pedida anteriormente deve ser renunciado. Eu cedo este para a Sangha.’ A Sangha deve ... você deve devolver ... Este tecido é renunciado, e então retornado: ‘Eu devolvo este tecido para este venerável.’”

Permutações

Se ele arranja que o tecido seja feito e ele assim o percebe, ele comete uma ofensa nissaggiya pacittiya, que implica a confissão com renúncia. Se ele arranja que o tecido seja feito mas ele não tem certeza se este foi feito, ele comete uma ofensa nissaggiya pacittiya, que implica a confissão com renúncia. Se ele arranja que o tecido seja feito mas percebe que este não foi feito, ele comete uma ofensa nissaggiya pacittiya, que implica a confissão com renúncia.

Se ele não arranjou que o tecido fosse feito mas ele percebe que este foi feito, ele comete uma ofensa dukkata, de transgressão. Se ele não arranjou que o tecido fosse feito mas ele não tem certeza, ele comete uma ofensa dukkata, de transgressão. Se ele não arranjou que o tecido fosse feito e assim o percebe, não há ofensa.

Quando não é uma ofensa

Não há ofensa quando: a linha é usada para se costurar um manto; a linha é para se fazer uma bandagem; a linha é para se fazer um cinto; a linha é para se fazer uma alça; a linha é para se fazer uma bolsa para tigela de esmolas; a linha é para se fazer uma peneira ou filtro d’água; se a linha é oferecida por seus parentes; se a linha é oferecida por quem faz um convite; se a linha é recebida em nome de outrem; se a linha é obtida através de suas posses; o indivíduo se encontra louco; ele é o primeiro a cometer a ofensa.

A sexta regra — sobre pedir por linha de costura — se encerra aqui.
