Saṃyutta Nikāya 37.2

Purisa Sutta

Uma Pessoa

“Bhikkhus, quando um homem possui cinco características ele é extremamente desagradável para uma mulher. Quais cinco? Ele não é belo, não é rico, não é virtuoso; ele é letárgico; e ele não gera bebês. Quando um homem possui essas cinco características ele é extremamente desagradável para uma mulher.

“Bhikkhus, quando um homem possui cinco características ele é extremamente agradável para uma mulher. Quais cinco? Ele é belo, rico, virtuoso; ele é hábil e laborioso; e ele gera bebês. Quando um homem possui essas cinco características ele é extremamente agradável para uma mulher.”
