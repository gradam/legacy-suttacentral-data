Saṃyutta Nikāya 22

15. Diṭṭhivagga

150. Ajjhattasutta

Sāvatthi­nidānaṃ. “Kismiṃ nu kho, bhikkhave, sati, kiṃ upādāya uppajjati ajjhattaṃ sukhadukkhan”ti? Bhagavaṃmūlakā no, bhante, dhammā … pe … “rūpe kho, bhikkhave, sati rūpaṃ upādāya uppajjati ajjhattaṃ sukhadukkhaṃ. Vedanāya sati … pe … saññāya sati … saṅkhāresu sati … viññāṇe sati viññāṇaṃ upādāya uppajjati ajjhattaṃ sukhadukkhaṃ. Taṃ kiṃ maññatha, bhikkhave, rūpaṃ niccaṃ vā aniccaṃ vā”ti? “Aniccaṃ, bhante”. “Yaṃ panāniccaṃ dukkhaṃ vā taṃ sukhaṃ vā”ti? “Dukkhaṃ, bhante”. “Yaṃ panāniccaṃ dukkhaṃ vipari­ṇāma­dhammaṃ, api nu taṃ anupādāya uppajjeyya ajjhattaṃ sukhadukkhan”ti? “No hetaṃ, bhante”. “Vedanā … pe … saññā … saṅkhārā … viññāṇaṃ niccaṃ vā aniccaṃ vā”ti? “Aniccaṃ, bhante”. “Yaṃ panāniccaṃ dukkhaṃ vā taṃ sukhaṃ vā”ti? “Dukkhaṃ, bhante”. “Yaṃ panāniccaṃ dukkhaṃ vipari­ṇāma­dhammaṃ, api nu taṃ anupādāya uppajjeyya ajjhattaṃ sukhadukkhan”ti? “No hetaṃ, bhante”. “Evaṃ passaṃ … pe … nāparaṃ itthattāyāti pajānātī”ti.

Paṭhamaṃ.
