Aṅguttara Nikāya 3

8. Ānandavagga

76. Paṭha­ma­bhava­sutta

Atha kho āyasmā ānando yena bhagavā tenupasaṅkami; upasaṅkamitvā bhagavantaṃ abhivādetvā ekamantaṃ nisīdi. Ekamantaṃ nisinno kho āyasmā ānando bhagavantaṃ etadavoca: “bhavo, bhavoti, bhante, vuccati. Kittāvatā nu kho, bhante, bhavo hotī”ti?

“Kāma­dhātu­ve­pakkañca, ānanda, kammaṃ nābhavissa, api nu kho kāmabhavo paññāyethā”ti? “No hetaṃ, bhante”. “Iti kho, ānanda, kammaṃ khettaṃ, viññāṇaṃ bījaṃ, taṇhā sneho. Avijjā­nīvara­ṇā­naṃ sattānaṃ taṇhā­saṃ­yoja­nā­naṃ hīnāya dhātuyā viññāṇaṃ patiṭṭhitaṃ evaṃ āyatiṃ punabbha­vā­bhi­nib­batti hoti. ()

Rūpa­dhātu­ve­pakkañca, ānanda, kammaṃ nābhavissa, api nu kho rūpabhavo paññāyethā”ti? “No hetaṃ, bhante”. “Iti kho, ānanda, kammaṃ khettaṃ, viññāṇaṃ bījaṃ, taṇhā sneho. Avijjā­nīvara­ṇā­naṃ sattānaṃ taṇhā­saṃ­yoja­nā­naṃ majjhimāya dhātuyā viññāṇaṃ patiṭṭhitaṃ evaṃ āyatiṃ punabbha­vā­bhi­nib­batti hoti. ()

Arūpa­dhātu­ve­pakkañca, ānanda, kammaṃ nābhavissa, api nu kho arūpabhavo paññāyethā”ti? “No hetaṃ, bhante”. “Iti kho, ānanda, kammaṃ khettaṃ, viññāṇaṃ bījaṃ, taṇhā sneho. Avijjā­nīvara­ṇā­naṃ sattānaṃ taṇhā­saṃ­yoja­nā­naṃ paṇītāya dhātuyā viññāṇaṃ patiṭṭhitaṃ evaṃ āyatiṃ punabbha­vā­bhi­nib­batti hoti. Evaṃ kho, ānanda, bhavo hotī”ti.

Chaṭṭhaṃ.
