Saṃyutta Nikāya

11. Khotbah yang Berhubungan dengan Sakka

6. Sarang-sarang Burung

Di Sāvatthī. “Para bhikkhu, Suatu ketika di masa lampau para deva dan para asura sedang bersiap-siap untuk suatu pertempuran. Dalam peperangan itu para asura menang dan para deva kalah. Dalam kekalahan itu para deva mundur ke utara sedangkan para asura mengejar mereka. Kemudian Sakka, raja para deva, berkata kepada kusirnya Mātali dalam syair:

“‘Hindari, O Mātali, dengan tiang keretamu
Sarang-sarang burung dalam hutan-hutan pohon kapuk;
Biarlah kita menyerahkan hidup kita kepada para asura
Daripada membuat burung-burung ini kehilangan sarang.’

“’Baik, Baginda’, Mātali si kusir menjawab, dan ia memutar balik keretanya bersama dengan barisan seribu ekor kuda berdarah murni.

“Kemudian, para bhikkhu, para asura itu berpikir: ‘Sekarang kereta Sakka dengan barisan seribu kuda berdarah murni berbalik. Para deva akan menghadapi pertempuran dengan para asura untuk ke dua kalinya.’ Diserang oleh ketakutan, mereka memasuki kota para asura. Demikianlah, para bhikkhu, Sakka, raja para deva, menang hanya dengan kebajikan.”

Translated by Indra Anggara for Dhamma Citta.
