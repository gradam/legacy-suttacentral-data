Saṃyutta Nikāya

11. Khotbah yang Berhubungan dengan Sakka

23. Kegaiban

Di Sāvatthī. Sang Bhagavā berkata sebagai berikut: “Para bhikkhu, suatu ketika di masa lampau Vepacitti, raja para asura, sedang sakit, menderita, sakit parah. Kemudian Sakka, raja para deva, mendatangi Vepacitti untuk menanyakan tentang penyakitnya. Dari jauh Vepacitti melihat kedatangan Sakka dan berkata kepadanya: ‘Sembuhkan aku, raja para deva.’—‘Ajari aku, Vepacitti, kegaiban Sambari.’—‘Aku tidak akan mengajarkannya, Tuan, hingga aku mendapat izin dari para asura.’

“Kemudian, para bhikkhu, Vepacitti, raja para asura, bertanya kepada para asura: ‘Bolehkah aku mengajarkan kegaiban Sambari kepada Sakka, raja para deva?’—‘Jangan ajarkan kegaiban Sambari kepadanya, Tuan.’

“Kemudian, para bhikkhu, Vepacitti, raja para asura, berkata kepada Sakka, raja para deva, dalam syair:

“Seorang penyihir—O Maghavā, Sakka,
Raja para deva, suami Sujā–
Pergi ke neraka yang mengerikan,
Bagaikan Sambara, selama seratus tahun.’”
