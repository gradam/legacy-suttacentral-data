Saṃyutta Nikaya 14

Von den Elementen

6. Element

Ort der Begebenheit: Sāvatthī.

„Ich will euch, ihr Bhikkhus, die Verschiedenheit der Elemente lehren; höret zu usw. usw. (= 1). ..

Welches ist aber, ihr Bhikkhus, die Verschiedenheit der Elemente? Das Element Form, das Element Ton, das Element Geruch, das Element Geschmack, das Element Fühlbarer Gegenstand, das Element Ding.

Dies, ihr Bhikkhus, heißt Verschiedenheit der Elemente.“
